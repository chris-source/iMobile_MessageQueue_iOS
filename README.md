# iMobile_MessageQueue_iOS
该应用基于SuperMap iMobile 8C for iOS 构建，提供多媒体数据照片、音频、视频的采集，绘制点标绘符号、箭头标绘符号，手势绘制线、面几何对象以及涂鸦功能。同时提供从移动端通过消息总线向其他端发送文本消息、位置数据、多媒体数据、标绘符号，手势绘制和涂鸦得到的几何对象。提供定位功能，实现位置共享。
